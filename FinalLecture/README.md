- **RLHF and Alignment**

## Usage

Each folder contains:
- `generate_*.py` - Python script to generate the chart
- `*.pdf` - Generated chart PDF
- `metainfo.txt` - Quantlet metadata

To regenerate a chart:
```bash
cd <folder>
python generate_*.py
```
