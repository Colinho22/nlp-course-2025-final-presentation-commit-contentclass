N-gram Language Models

Statistical Foundations of NLP

Overview

Learn how to predict the next word using statistical patterns from text. N-gram models form the mathematical foundation for understanding language probability and serve as the baseline for all modern language models.

Learning Objectives

  • Calculate conditional probabilities P(word|context) from corpus statistics
  • Implement smoothing techniques (Laplace, Add-k) to handle unseen n-grams
  • Evaluate language models using perplexity metrics
  • Understand the trade-offs between model complexity and data sparsity
